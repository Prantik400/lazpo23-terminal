# ⚡ LAZP023 Terminal Dashboard

A futuristic **sci-fi inspired terminal dashboard UI** built using pure frontend technologies.  
Designed to simulate real-time system monitoring with animated components, alert systems, and interactive controls.

---

## 🧠 Overview

LAZP023 Terminal is a visually rich dashboard that mimics a **system control interface**, featuring:

- CPU & GPU animated status indicators
- Real-time error tracking UI
- Interactive command terminal panel
- Alert system with visual + audio feedback
- Live error graph visualization

This project focuses on **UI engineering, animations, and event-driven behavior**.

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Layout, animations, neon UI
- **JavaScript (Vanilla)** – Logic & interactivity
- **Chart.js** – Error graph visualization

---

## ✨ Features

### ⚙️ Core System UI

- Animated CPU & GPU indicators using CSS animations
- Dynamic system stats display (errors, warnings, success)

### 🚨 Error Detection System

- Blinking neon red error indicator
- Continuous alert sound on anomaly detection
- Global screen glow effect during critical alerts
- Auto-clear on reconnect/close actions

### 🎛️ Command Control Panel

- **ON** → Establish connection (simulated)
- **RECONNECT** → Reset system state
- **CLOSE** → Terminate connection
- Button animations with visual feedback

### 📊 Error Graph

- Real-time line chart using Chart.js
- Dynamic updates through JavaScript

---

## 🖥️ UI Highlights

- Fully custom **sci-fi neon design**
- Responsive 3-column layout (40% / 30% / 30%)
- Pixel-aligned bordered panels
- Smooth animations + glow effects
